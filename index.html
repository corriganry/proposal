Application Overview

This application will allow you to convert Binary to Decimal and Decimal to Binary.

It will have an aesthetically pleasing interface and will make it easy to take part in the conversions.

It will also keep track of past conversions that take place in a database.

Technology Overview

I will be using Visual Studio Code to program. I will be using HTML, CSS, JavaScript, and MongoDB. I will be hosting this project on Azure.

Feature List

  • Conversion of Binary to Decimal
  • Conversion of Decimal to Binary
  • Being able to look at the last 5 conversions that took place
  • Instructions on how to convert yourself

Future Feature List

  • Conversion of Binary to Hexadecimal
  • Conversion of Hexadecimal to Binary

Basic Technical Features

  • I will be using HTML to make a webpage that allows me to take in numbers to be converted.
  • I will be using CSS to style this page and make it look aesthetically pleasing.
  • I will be using JavaScript to perform functions that allow me to make the proper conversions.
  • Lastly, I will be using MongoDB to store these conversions in a database that allows the user to see past attempts
